# datable  
>maybe the best open source alternative to excel.  

 :warning:  datable is under development, far from being stable to be used yet. 

## target

- powerful spreadsheet backend

## dev 
```sh
 git clone https://github.com/uptonking/datable.git
 cd datable/
 ./start-build-dev.sh
```
